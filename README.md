```

 ┣ assets       // svg 파일
 ┣ components   // 공통으로 사용하는 컴포넌트
 ┣ hooks        // Custom Hooks
 ┣ routes       // 페이지
 ┣ services     // API 호출 관련
 ┣ store        // 전역 상태
 ┣ styles       // 전역 style
 ┣ types        // 필요한 type 정의
```
## 📍 Getting Started / 어떻게 시작하나요?
1. Repository 클론
```sh
$ git clone git@github.com:Team8-Rocket/madup-8B.git
```

2. Dependecies 설치
```sh
$ yarn install
```

3. Run 실행
```sh
$ yarn start
```
